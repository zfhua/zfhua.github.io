vim常用快捷键

Jan 1, 2018

移动图标示意图

gg:    文件头
G:    文件尾
nG: 跳转之n行

0:    行首
$:    行尾

Vim 常用图示Link

    1G/gg/Home
        k
^/0  h 光标 l  $
        j
    G/End
复制/剪切/粘贴 / 查询
*复制/粘贴*:
yy / p 复制一行 n+yy 多行
dd / p 剪切一行 n+dd 多行
*查找:
:/word ( :?word ) 继续按 n/N
*定位*:
:行号

撤销操作：u
还原撤销：ctrl + r
v：进入 visual 模式
    d : 剪切操作
    y : 选择复制
    p : 粘贴操作
ctrl + v：进入 visual block 模式
    ...
设置行号
:set number
文件配置自动显示行号
echo "se nu" >> ~/.vimrc
中文乱码
~/.vimrc --> 添加 -->
:set fileencodings=ucs-bom,utf-8,cp936,gb18030,big5,euc-jp,euc-kr,latin1
注:
对于类似 $name 命名的文件,删除需:
rm '$name'
