利用Hexo 制作博客

####简明教程

比较流行的一些静态博客框架有Jekyll、Hexo、FarBox、Simple、Octopress、Pelican等等；

我准备的是 Hexo 教程：

  • 环境准备
  1. Git
  2. Node.js
  3. Hexo

Hexo官方中文文档 Hexo
NexT主题开发文档 NextT

  • 安装 Hexo

> npm install -g hexo-cli

  • 初始化博客空间

> hexo init

  • 安装博客项目node环境依赖

> npm install

目录结构：

  • 生成静态文件

> hexo -g 或 hexo generate

  • 启动本地服务预览效果

> hexo -s 或 > hexo server

  • 新建文件
    > hexo new

  • 上传部署博客

  • 配置文件 _config.yml 参数说明
